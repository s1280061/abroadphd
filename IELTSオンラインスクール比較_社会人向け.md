# 🏫 社会人のためのIELTSオンラインスクール比較

働きながらIELTS（Academic）を攻略する社会人向けに、オンライン対応のスクール／講座を**タイプ別**に整理しました。
[IELTS勉強法_社会人向け.md](IELTS勉強法_社会人向け.md) の「添削への投資」の具体的な選択肢編です。

> ⚠️ **重要な免責**
> 料金・コース・保証条件は**頻繁に変わります**。本ページの金額は各比較サイト（2025〜2026年時点）の記載に基づく**目安**で、正確な情報は必ず**各スクール公式サイト＋無料カウンセリング**で確認してください。特定サービスの宣伝・保証ではありません。

## 1. 社会人の選び方：4つの軸

1. **弱点技能はどこか** … Writing/Speakingが弱い→**添削・マンツーマン型**。Listening/Reading中心→独学＋問題集でも可。
2. **自己管理できるか** … 一人だと続かない→**コーチング型**（学習管理つき）。自走できる→オンライン英会話＋添削で十分。
3. **予算** … 月数千円のオンライン英会話〜3ヶ月60万円超のコーチングまで**桁が違う**。総額（入会金・教材込み）で比較。
4. **期限** … 締切まで短い→**短期集中・スコア保証型**。時間に余裕→安価に長く。

**社会人の定石**: 「時間 > お金」なら**お金で時間を買う**（コーチング/添削）。まずは**無料カウンセリング/体験**を2〜3社受けて相性を見る。

## 2. タイプ別おすすめ

### 🅰 IELTS特化オンライン（添削 × スピーキング）— まず検討したい本命
Writing/Speakingを効率よく伸ばしたい社会人の第一候補。

| サービス | 特徴 | 向いている人 |
|----------|------|--------------|
| **PlusOnePoint（プラスワンポイント）** | IELTS試験対策に特化。IDP Educationの講師研修を修了した**日本人講師＋日英バイリンガル/ネイティブ講師**。全てオンライン完結。1レッスンで**Writing30分＋Speaking30分**などカスタム可 | 日本語での論理的解説が欲しい／W・Sを集中的に |
| **Best Teacher（ベストティーチャー）** | **英作文→添削→その英文でスピーキング**という独自フロー。IELTS採点基準に基づくエッセイ添削 | W・Sを一体で鍛えたい／自分の書いた英語で話す練習 |
| **Kumikoのオンライン講座** | 専門講師による**4技能の個別添削**。目安スコアつきフィードバック、受講中は質問無制限 | 添削中心で客観評価が欲しい |

### 🅱 短期集中・コーチング型（時間管理＋スコア保証）— 締切が近い/自走が苦手な人
高額だが「学習管理」ごと外注でき、**保証制度**を持つところも。

| サービス | 特徴 | 料金の目安（要確認） |
|----------|------|----------------------|
| **PROGRIT（プログリット）** | 3ヶ月の完全パーソナルトレーニング。専任コーチが学習管理、IELTSコースは**スコア8.0監修**。アプリで進捗管理 | 約**665,500円**/3ヶ月 |
| **TORAIZ（トライズ）** | コーチング型。**0.5点未達なら1ヶ月無料延長＋全額返金保証**などの制度 | 要確認（コーチング相場） |
| **English Innovations（イングリッシュイノベーションズ）** | **点数保証制度**（未達なら無料で追加受講）。短期スコアアップ実績を掲げる | 約**240,000円** |
| **Liberty English Academy** | 独自メソッド「**グラマーテーブル**」。成果保証、6.5以上の高スコア狙い | 約**336,000〜792,000円**（回数による） |

### 🅲 大手・留学サポート一体・公式系 — 出願まで伴走してほしい人
| サービス | 特徴 |
|----------|------|
| **British Council** | IELTS共同運営団体の公式対策（約**174,240円**の講座例）。信頼性重視 |
| **バークレーハウス** | IELTS公式（IDP）団体。試験対策の実績校 |
| **アゴス・ジャパン** | 海外大学・大学院進学支援と一体。**有効期間3年・再受講**などサポート厚め |
| **ベルリッツ／日米英語学院** | マンツーマン＋オンライン対応。事前にスコア・弱点分析。初心者コースあり |

### 🅳 オンライン英会話（安価・話す量を稼ぐ）— 予算重視/Speaking量が課題
IELTS特化度はAより低いが、**低価格で発話量を確保**できる。独学＋公式問題集と併用向き。

| **DMM英会話** | IELTS対策教材あり | 教材が豊富・コスパ良 |
| **QQ English** | フィリピン人講師でIELTS対策 | 体系的カリキュラム |

## 3. 料金の桁感（総額で考える）

| 層 | 目安（総額/月額） | 例 |
|----|-------------------|----|
| 低コスト（自走＋発話量） | 月**7,000〜10,000円**前後 | NativeCamp・DMM英会話・ユニバーサルスピーキング |
| 中コスト（特化添削/大手） | 総**17〜25万円**前後 | British Council・English Innovations・日米英語学院 |
| 高コスト（コーチング/保証） | 3ヶ月**33〜80万円** | PROGRIT・Liberty・（TORAIZ） |

※いずれも**入会金・教材費込みの総額**で比較を。安さだけでなく**保証条件（対象スコア・回数）**も要チェック。

## 4. 社会人向け・選び方フロー（早見）

- **Writing/Speakingで詰まっている** → 🅰（PlusOnePoint / Best Teacher / Kumiko）
- **一人だと続かない・締切が近い・確実性重視** → 🅑（PROGRIT / TORAIZ / English Innovations / Liberty）
- **出願校選び〜エッセイまで一括で頼りたい** → 🅒（アゴス / バークレーハウス / British Council）
- **予算を抑えて発話量を稼ぎたい／自走できる** → 🅓（NativeCamp / DMM / QQ English）＋ Cambridge公式問題集で独学

**組み合わせが現実解**: 例）独学（公式問題集で L/R）＋ オンライン英会話（S量）＋ 特化スクールで W/S添削、を並行。

---

## 5. 契約前チェックリスト
- [ ] **無料カウンセリング/体験**を2〜3社受けて講師・解説の相性を確認したか
- [ ] **総額**（入会金・教材費込み）で比較したか
- [ ] **保証制度の条件**（対象スコア・回数・返金要件）を読んだか
- [ ] 自分の**弱点技能**にそのスクールが強いか（W/S添削の質）
- [ ] オンラインの**受講可能時間帯**が自分の生活（平日夜・週末）と合うか
- [ ] 解約・休会の条件（社会人は繁忙期に中断したくなる）

## 6. 事例集とのつながり
- 本リポジトリの当事者は **IELTS 5.5→6.5・TOEFL54→IELTS6.5** など、出発点が高くない人も合格（[Kazuki](海外/退職して博士/Kazuki_一輝.md)・[Inoue](海外/退職して博士/Inoue.md)）。→ **必要ラインを最短で超える**発想でスクールを"手段"として使う。
- 独学組ほど「英語で苦労」と語る一方、**添削・会話量の確保**でカバーしている。→ W/Sは🅰🅑の投資が効く。
