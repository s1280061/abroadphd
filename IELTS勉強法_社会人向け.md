# 🎯 社会人のためのIELTS勉強法

海外大学院（修士・PhD）や奨学金の出願で必要になる **IELTS（Academic）** を、**働きながら**攻略するための実践ガイド。
本リポジトリの[事例集](README.md)の当事者たちの声（例: Kazuki=IELTS 5.5→6.5、Inoue=TOEFL54→IELTS6.5、中島記念財団=IELTS7.0が要件）も踏まえてまとめています。

> ⚠️ 免責: IELTSの制度は変わり得ます。出願先の要求スコアと最新の試験仕様は必ず[公式（IELTS/British Council/IDP）](#参考url)で確認してください。

---

## 0. まず「敵」を知る（30分で全体像）

| 項目 | 内容 |
|------|------|
| 種類 | 大学院出願は **Academic** モジュール（General ではない） |
| 4技能 | Listening / Reading / Writing / Speaking |
| スコア | 各セクション **0〜9.0**（0.5刻み）。Overall は4技能の平均を0.5単位に丸める |
| 形式 | 会場受験（紙/PC）と **IELTS on computer**、自宅受験の一部。Speakingは試験官と対面/オンライン |
| 有効期限 | 2年間 |

### 出願で必要なスコアの目安（要・各校確認）
| 目的 | Overall の目安 | 補足 |
|------|----------------|------|
| 一般的な大学院（修士/PhD）最低ライン | **6.5**（各6.0以上を課す学校が多い） | Writingだけ7.0要求など**各技能の最低点**に注意 |
| トップ校・人文社会系・奨学金 | **7.0〜7.5**（各6.5〜7.0） | 中島記念国際交流財団は英検1級/TOEFL90/**IELTS7.0**のいずれか |
| 「あと0.5」が一番遠い | — | 特に **Writing と Speaking** は6.5→7.0の壁が厚い |

**ポイント**: 「Overallさえ届けばOK」ではなく、**足切り（各技能の最低点）**で落ちる人が多い。弱点技能を放置しない。

---

## 1. 社会人が最初にやること：逆算とスコア診断

### (1) 期限から逆算する
- スコアを **1.0上げるのに約3ヶ月・合計120〜180時間**（週10〜15時間）が一般的な目安。
- 出願締切（多くは秋〜冬）から逆算し、**「受験日」を先に2〜3回分予約**してしまう（人は締切がないと動かない）。
- Speaking/Writingは伸びが遅いので**早めに着手**。Listening/Readingは型を覚えると比較的速く伸びる。

### (2) 現状を測る（最初の週末に1回）
- **Cambridge 公式問題集（IELTS 18〜21 など）**を1セット、時間を計って解く。
- 出た「素点→バンド換算」で現在地を把握し、**目標との差**と**最弱の技能**を特定する。
- これをやらずに教材を買い漁るのが**社会人の典型的な失敗**。

---

## 2. 「時間がない」を解くのが社会人IELTSの本質

事例の当事者もほぼ全員が「**時間確保**」を最大の壁に挙げています。時間を"作る"より"埋め込む"。

| スキマ | 使い方 |
|--------|--------|
| 通勤（往復） | Listening のシャドーイング / 単語アプリ |
| 昼休み | Reading 1パッセージ（20分） |
| 平日夜 60〜90分 | Writing 1本 or 弱点補強（週3〜4回） |
| 週末 半日×1 | **本番形式のフル模試**＋復習（超重要） |

- **平日は"維持と積み上げ"、週末に"本番演習"**、が働きながらの黄金パターン。
- 「毎日ゼロにしない」。5分でも単語・リスニングを触れて習慣を切らさない。
- 有給・繁忙期を見て、**試験2週間前は負荷を上げられる週**を確保しておく。

## 3. セクション別・攻略法

### 🎧 Listening（40問／約30分＋転記時間）
- **毎日20分**、IELTS音声を聞く習慣。特に **数字・日付・スペル・固有名詞** の聞き取りを重点練習。
- **ディクテーション**（書き取り）と**シャドーイング**が最強。聞き取れない箇所＝伸びしろ。
- 設問を**先読み**して「何を待ち受けるか」を決めてから聞く。
- イギリス英語・豪州英語など**多様なアクセント**に慣れる（BBC等）。

### 📖 Reading（40問／60分）
- 全部を読まない。**Skimming（大意）＋Scanning（該当語を探す）**。
- **時間配分**が命。1パッセージ20分、難しい設問は捨てて先へ（後で戻る）。
- 頻出設問（True/False/Not Given、見出しマッチング、穴埋め）ごとに**解き方の型**を固定。
- **Not Given と False の区別**が鬼門。「本文に書いていない＝Not Given」を徹底。

### ✍️ Writing（Task1 20分＋Task2 40分）
Academicは **Task1＝図表の説明**、**Task2＝エッセイ（配点大）**。評価は4基準（各25%）:
1. **Task Response**（設問に完全に答え、立場が明確か）
2. **Coherence & Cohesion**（論理構成・つなぎ）
3. **Lexical Resource**（語彙の幅と正確さ）
4. **Grammatical Range & Accuracy**（文法の幅と正確さ）

Band7の要点:
- **必ず最初に3〜5分プランを書く**。構成（序論→本論2つ→結論）を決めてから書く。
- **立場（意見）を最初から最後まで一貫**させる。全文がその主張に奉仕する。
- Task2を**Task1より優先**（配点が高い）。時間切れで結論なしは大幅減点。
- **テンプレ表現＋自分の語彙**。丸暗記の"型"だけでは7.0は出にくい。
- **独りでは伸びない技能No.1**。→ 添削（後述）必須。

### 🗣️ Speaking（11〜14分／Part1雑談・Part2スピーチ・Part3議論）
- **スマホで録音→文字起こし→自己添削**。詰まり・文法ミス・語彙の単調さを客観視。
- Part2は**1分準備→2分話す**。日常トピックを**"自分の実体験"に紐づけて**話すネタ帳を作る（社会人は実務経験が強い武器）。
- 流暢さ＞完璧さ。**止まらない**ことが評価される。言い換え（paraphrase）力を鍛える。
- **オンライン英会話**で「話す量」を確保するのが社会人には効率的。

## 4. 教材（増やしすぎない・薄く広くしない）

| 種類 | 定番 | 使い方 |
|------|------|--------|
| **公式問題集（最重要）** | Cambridge IELTS 18〜21 等 | 本番形式の演習と素点→バンド換算。"赤本"の位置づけ |
| 単語 | 実践IELTS英単語3500 / Cambridge Vocabulary for IELTS | 毎日5〜10語。すきま時間はアプリで |
| 文法/瞬発力 | 英語のハノン 等 | Speaking/Writingの土台。パターン練習 |
| 弱点別 | セクション別対策本（Writing/Speaking特化 等） | 最弱技能にだけ足す |
| アプリ | 単語・リスニング系 | **補助**。高得点は公式問題集が主 |

**原則**: 公式問題集を軸に据え、参考書は**最弱技能に1冊足す**程度。教材ジプシーは時間の敵。

---

## 5. Writing / Speaking は「添削」で伸ばす（社会人こそ課金）

- 自分では**改善点に気づけない**のがWriting/Speaking。**第三者フィードバック**が費用対効果最強。
- 選択肢: オンライン英会話のIELTS対策コース／IELTS添削サービス／ネイティブ講師の単発添削。
- 社会人は「時間>お金」になりがち。**お金で時間を買う**（＝添削・スクール）判断が合理的。
- 事例でも、独学の人ほど「英語で苦労」と語る一方、教授コンタクトや実務経験でカバーしている。

## 6. 週次スケジュール例（働きながら／週12時間・3ヶ月）

| 曜日 | 内容（目安） |
|------|--------------|
| 月 | 夜60分：Writing Task2を1本→（翌日以降に添削へ） |
| 火 | 通勤＋夜：Listening演習＋ディクテーション30分／単語 |
| 水 | 夜60分：Reading 2パッセージ（時間計測） |
| 木 | オンライン英会話30分（Speaking Part2/3）＋録音復習 |
| 金 | 軽め：単語・リスニングで習慣維持（20〜30分） |
| 土 | **フル模試（本番同形式）**＋徹底復習（2〜3時間） |
| 日 | 弱点補強＋前週の添削フィードバック反映（1〜2時間） |

- 3ヶ月を **月1: 型を覚える → 月2: 弱点集中 → 月3: 本番演習と微調整** に分ける。
- 直近2週間は**本番と同じ時間帯・同じ順序**で通し練習し、体を慣らす。

---

## 7. 試験直前・当日のコツ
- 直前1週間は**新しい教材を増やさない**。既出の型・単語・自分の弱点ノートを回す。
- **時間配分の型を体に入れる**（Reading各パッセージ20分、Writing Task1=20分/Task2=40分）。
- Speakingは前日に**"よく出る質問→自分の実体験ネタ"**を音読して口を慣らす。
- 当日は**Listeningの先読み**と**Writingのプランニング**を必ず実行。

---

## 8. 社会人が陥りやすい失敗
1. **診断せずに教材を買う** → まず公式問題集で現在地を測る。
2. **Overallだけ見て足切り技能を放置** → 最低点要件を必ず確認。
3. **Writing/Speakingを独学で放置** → 添削・会話量の確保に投資。
4. **平日ゼロ→週末だけ** → 習慣が切れる。毎日5分でも触れる。
5. **受験を先延ばし** → 先に受験日を予約。1回で決めず**複数回**前提で。
6. **完璧主義**（特にSpeaking） → 止まらない・言い換える方が高得点。

---

## 9. この事例集から学べること（IELTS×社会人）
- **5.5→6.5、TOEFL54→IELTS6.5** など、**出願時点で英語が高くない人も合格**している（[Kazuki](海外/退職して博士/Kazuki_一輝.md)・[Inoue](海外/退職して博士/Inoue.md)）。
- 合否は英語スコアだけでなく、**教授への直接コンタクト・研究業績・実務経験**の比重が大きい（[summary.md](summary.md) 3.7参照）。
- 出願を決めてから**短期集中でスコアを作る**人が多い（例: [三宮さん](海外/社会人を続けながら博士/三宮.md)は1ヶ月でIELTS対策）。
- つまり **「英語で完璧を待たず、必要ラインを最短で超えて出願する」** のが社会人の勝ち筋。
